Display custom messages on administrator screen based on user roles.

== Description ==
Author: [Small Business Web Design Sydney](http://www.smallbusinesswebdesigns.net.au "Small Business Web Design Sydney")

Plugin Link: http://www.smallbusinesswebdesigns.net.au/wordpress-admin-screen-messages-release.html

Description: This feature allows you to set custom messages at the top of administrator screen, for users based on their role. 

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/admin-screen-messages/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to settings -> Admin Messages option page (users with administrator roles only can see the options page ).
4. Add your custom messages for each roles

== Frequently Asked Questions ==
= How to install the plugin? =
Download the .zip file from [plugin repository](https://downloads.wordpress.org/plugin/admin-screen-messages.zip) website and Upload the plugin into your WordPress website. Then, Install the plugin and activate the plugin. You can also install the plugin through the WordPress plugins screen directly.
= How can i add notice or messages in administrator screen for subscribers? =
Go to the Admin Screen Messages option page under Settings, then add your message for Subscriber field in "Add custom message for Role : Subscriber" section.
= How can i add notice or messages in administrator screen for subscribers? =
Go to the Admin Screen Messages option page under Settings, then add your message for Administrator field in "Add custom message for Role : Administrator" section
